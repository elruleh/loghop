## Security

For security vulnerabilities, please report privately:
[GitHub Security Advisories](https://github.com/elruleh/loghop/security/advisories/new)

**Do not file public issues for security vulnerabilities.**

## Response Time

- **Bugs:** We aim to respond within 48 hours
- **Feature requests:** We review periodically
- **Security:** Critical issues resolved within 7 days

> Note: `loghop` is maintained by a single person in their spare time.
> Issues and discussions are answered best-effort. For anything urgent
> (security, abuse, takedown), use the private channel above.

## Common Issues

### Installation Problems

See [Getting Started](docs/getting-started.md) for installation instructions.

### Session Capture Issues

See [Troubleshooting](docs/how-to/troubleshooting.md) for common problems and solutions.

### Provider Not Found

Run `loghop providers` to see which providers are detected. Ensure the provider binary is on your PATH.

## Getting Started

1. Install: `pipx install loghop`
2. Initialize: `loghop init`
3. Run: `loghop run`

Full guide: [Getting Started](docs/getting-started.md)
